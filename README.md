# AI-Enabled Candidate Resume Screening

## Overview
This project is an **AI-powered tool** designed to streamline the process of screening candidate resumes. It automatically analyzes resumes, extracts key information, and ranks candidates based on predefined criteria, helping HR teams save time and improve hiring decisions.

## Features
- Resume parsing and information extraction (e.g., skills, experience, education)
- Candidate ranking based on relevance and qualifications
- Support for multiple file formats (PDF, DOCX)
- Easy-to-use interface or script-based workflow
- Modular design for easy integration with existing HR systems

## Tech Stack
- **Programming Language:** Python
- **Libraries/Frameworks:** TensorFlow, Keras, Pandas, NumPy, Flask
- **Other Tools:** Git, GitHub
